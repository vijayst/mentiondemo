# dandi-ui
Dandi UI - React app

To run locally:
* yarn install
* yarn start
* Open http://localhost:8080


To deploy to dev:
* yarn build:dev
* yarn deploy:dev
* Open https://dandi-dev.firebaseapp.com

To deploy to prod:
* yarn build:prod
* yarn deploy:prod
* Open https://itsdandi.com

Happy coding !!!
